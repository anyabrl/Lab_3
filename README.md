## Реализация принципов ООП и SOLID
Архитектура приложения спроектирована с четким разделением ответственности:
* **Инкапсуляция:** Данные строго скрыты внутри свойств моделей (`TemperatureRecord`, `CrimeRecord`), доступ к ним осуществляется только через getter/setter.
* **Полиморфизм:** Единый класс `ForecastService` применяется для расчета скользящей средней как для температуры, так и для преступности.
* **SOLID (Принцип единой ответственности - SRP):** Визуальные компоненты (`UserControl`) отвечают **только** за интерфейс. Вся бизнес-логика (математика, поиск минимумов/максимумов) вынесена в отдельные классы-сервисы (`TemperatureAnalyzer`, `CrimeAnalyzer`).
* **SOLID (Принцип инверсии зависимостей - DIP):** Формы не реализуют логику внутри кнопок, а делегируют выполнение специализированным сервисам.

## Вариант 3: Температура в городе (Участник 1)
Пользователь открывает файл с данными о температуре за месяц.
* **Таблица:** Вывод данных в удобном табличном формате (DataGridView).
* **Графики:** Зависимость от дня. Минимальная температура выделена **синим цветом**, максимальная — **красным**.
* **Аналитика:** Программа автоматически вычисляет дни, в которые перепад температуры был **самый сильный и самый слабый**.
* **Прогноз:** Экстраполяция методом скользящей средней на N дней (пользователь указывает N в текстовом поле). Прогнозные данные отображаются на графике **другим цветом (пунктир)**.
* **Интерактивность:** Графики поддерживают масштабирование (зум) колесиком мыши. Реализован **экспорт графиков в формат PNG**.

## Вариант 13: Преступность в России (Участник 2)
Пользователь открывает файл с данными о преступности в России за последние 15 лет.
* **Таблица:** Вывод статистики по годам в табличном формате.
* **Графики:** Построен график зависимости общего количества преступлений от года.
* **Аналитика:** Программа анализирует три вида преступлений и автоматически вычисляет, **какой вид преступности за 15 лет снизился больше всего, а какой меньше всего** (в процентах). Результат выводится в текстовую метку.
* **Прогноз:** Реализована экстраполяция скользящей средней на последующие N лет.
* **Интерактивность:** Графики интерактивные (зум), реализован **экспорт в PNG**.

## Как запустить проект
1. Клонировать репозиторий на локальный компьютер.
2. Открыть файл `Lab3.sln` в Visual Studio 2022.
3. Восстановить NuGet-пакеты (при необходимости).
4. Запустить приложение (F5).
5. На вкладке нужного варианта нажать "Открыть файл" и выбрать один из приложенных JSON-файлов с тестовыми данными.
6. Использовать кнопки построения прогноза и экспорта графика.
